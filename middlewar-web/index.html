LES RESSOURCES

RESSOURCES PRIMAIRES

Les ressources primaires servent à la construction des bâtiments, des unités de combat, des objets du jeu… Elles peuvent être récupérées par différents bâtiments.

CREDITS GALACTIQUES

Monnaie du jeu pour les échanges entre les joueurs

ÉNERGIE

Nécessaire au bon fonctionnement des bâtiments

OBJETS

Les objets sont une ressource supplémentaire, qui contrairement aux ressources primaires ne se produisent pas petit à petit avec le temps. Il faut passer commande dans les usines (bâtiment constructible dans une base), qui se mettent alors à produire la quantité demandée, dans la mesure des stocks de ressources ou objets disponible.

Les objets sont utilisés pour construire d’autres objets, des modules, des bâtiments, des unités de combat.

POPULATION

Présente sur les planètes, la population se développe. Elle a des attentes qui doivent être satisfaites. Elle paye des impôts, première source de revenue du joueur. Plus la population est élevée, plus les revenus sont importants. Plus la population est heureuse, plus les impôts peuvent être élevés. La population diminue si elle est maltraitée, la base est attaquée, ou lors de la construction de vaisseaux nécessitant des pilotes. Certains bâtiments ont besoin d’une main d’œuvre, occupant ainsi une partie de la population.

LES BASES

Le joueur commence avec une base sur une planète. Il sera amené à en créer d’autre, sur des planètes comme sur des lunes.

Les bases permettent de construire des bâtiments et de faire développer la population. Les opérations militaires peuvent être lancées depuis une base, ou depuis une orbite. Certains disent même que les plus grandes puissances sont capables de créer des bases directement dans l’espace… Permettant alors de se cacher des indiscrets, et surgir par surprise sur le champ de bataille.

LES BATIMENTS

Les bâtiments servent à débloquer différentes fonctionnalités pour une base ou pour un joueur. Ils peuvent être améliorés pour offrir de meilleures performances, ou débloquer d’autres fonctionnalités.

Certains bâtiments peuvent être équipés de modules. Si le bâtiment peut être équipé de module, il a alors un nombre maximal d’emplacements de modules. Les modules ayant une taille définie, s’il y a 2 emplacements de modules sur le bâtiment, le joueur peut y mettre soit 2 modules de taille 1, soit 1 module de taille 2.

LES MODULES

Les modules sont des ajouts qui peuvent être fait à des bâtiments. Il existe différent types de modules.

Sur un bâtiment de production de ressource, on peut ajouter par exemple un module d’amélioration de productivité augmentant la production, soit de manière proportionnelle, soit de manière brute.

Si un bâtiment produit une ressource de type fer au rythme de 1000 unités par heure, et qu’il y a 3 modules d’équipé sur le bâtiment :

  • Un module d’amélioration de productivité brut de 50 unités de fer par heure
  • Deux modules d’amélioration de productivité proportionnelle de 50% par heure

Dans ce cas les deux modules proportionnels se cumulent et offrent alors un bonus de 100% à la production de base. Les modules proportionnels n’influent pas sur les bonus de production brut. Ainsi, dans l’exemple donné la formule de la production est 1000 * (0.5 + 0.5) + 50 = 2050 unités de fer par heure.

Il existe différents types de modules améliorant :

  • Les coûts énergétiques
  • La capacité de stockage
  • La vitesse de production
  • La vitesse de construction
  • L’efficacité du bouclier
  • L’efficacité du contre-espionnage
  • …

Pour tous les modules, la stratégie de calcul est la même qu’expliquée précédemment.

Il existe d’autres types de modules pour les unités de combats. Voir la section correspondant plus bas.

LES UNITES DE COMBATS

Les unités de combats sont un assemblage d’objets que le joueur a choisis de mettre ensemble. À la base de toute unité de combat il y a une structure.

LES STRUCTURES

Les structures sont produites dans les usines comme tous les autres objets.

Tout comme les bâtiments, les structures disposent d’un certain nombre d’emplacements pour :

  • Des armes
  • Des moteurs
  • Des protections (boucliers, blindage…)
  • Des modules
  • Des technologies
  • Des cargos

LES MODULES DE COMBAT

Les modules de combat sont pour certains sur le même principe que les modules pour les bâtiments (amélioration de l’efficacité des armes, des moteurs, des boucliers…), et pour d’autre ils apportent des capacités unique aux unités de combat.

Quelques exemples :

  • Furtivité : rendant l’unité de combat plus difficile à détecter
  • Extraction : permettant à l’unité de combat d’extraire des ressources des OSNI
  • Exploration : permettant à l’unité de combat de scanner une planète ou une lune pour en découvrir quelques caractéristiques
  • Porte spatiale : permet l’ouverture d’une porte spatiale pour transporter une flotte discrètement et rapidement

L’UNIVERS

LES PLANETES

  • Température
  • Superficie
  • Quantité de ressources basiques infinie
  • Quantité de ressources rares limitées avec ou sans régénération
  • Nombre de lunes
  • Une orbite + vitesse de rotation
  • Type de planète (Définira ses bonus / malus)
  • Population
  • Humeur de la population

Les différentes caractéristiques d’une planète influeront sur le temps de trajet des unités de combat entre 2 planètes, si celes-ci ont besoin de descendre sur la planète pour agir. Il peut donc être intéressant et stratégique de posséder une lune pour accélérer les échanges commerciaux ou les départs de flottes.

LES LUNES

  • Taille de la lune
  • Ressource spéciale : Matière noire
  • Accès aux bâtiments spéciaux disponibles uniquement sur les lunes

Les lunes peuvent produire certaines ressources rares non disponibles sur les planètes, et peuvent servir de relai pour les flottes pour accélérer les échanges commerciaux ou les départs de flottes.

OSNI (OBJET SPATIAL NON IDENTIFIE)

Matière première gisant dans l’espace, résultat d’une confrontation entre deux flottes. Ils peuvent être analysés ou récupérés

ASTEROÏDES

Objet volant à vitesse réduite se déplaçant d’un point à un autre. Les vaisseaux ayant un module d’extraction peuvent se poser dessus et extraire des ressources. En maîtrisant bien les techniques de vol spatial, il peut être possible aux meilleurs pilotes de s’y cacher de manière à s’approcher de l’ennemi sans se faire repérer !

TROUS NOIR

Un grand mystère règne autour des trous noirs… à vous de résoudre l’énigme !

L’exploration dans Middlewar

Concept

Dans l’univers se trouvent des trous de vers, permettant de voyager vers des lieux inconnus appelés secteurs sombres. Les trous de vers ne sont pas ouvert indéfiniment. Ils finissent par se refermer au bout d’un certain temps (ouvert au maximum 24h). Afin de ne pas trop perdre le joueur, celui-ci peut savoir s’il reste s’il reste moins de 1,6,12 heures.

Il y a différents niveaux de secteurs sombres, plus le niveau est élevé, plus il faut traverser de secteur sombres pour le trouver (et donc autant pour retourner dans l’espace connu). Les secteurs sombres de niveau 1 sont accessible depuis l’univers connu, les secteurs sombres de niveau 2 uniquement à partir de secteurs niveau 1…

Pour assurer de pouvoir toujours revenir vers l’univers connu, il faut toujours qu’il y ait dans chaque secteur sombre un trou de ver vers un autre secteur de même niveau ou inférieur (on peut considérer un secteur de niveau 0 comme un secteur de l’univers connu). Dans le cas où un secteur niveau 1 perd son accès depuis l’univers connu, un nouvel accès s'ouvre vers un autre système de l’univers connu. Potentiellement très loin de l’entrée initiale.

Voyager dans les secteurs sombres

Le joueur peut décider de lancer une équipe d’exploration qui partira traverser le trou de ver. Pendant la traversée plus ou moins longue, la flotte ne serait donc plus détectable par les flottes ennemies. Une fois la traversée terminée, le joueur reçoit un rapport de ce qui est présent dans le secteur (plus ou moins détaillé en fonction des scanner présent sur les vaisseaux):

  • D’autres trous de vers
  • Des vestiges historiques (protégés ou non)
  • Des nuages de gaz qui peuvent être récoltés
  • Des astéroïdes offrant des ressources rares
  • Des flottes pirates

Le joueur peut alors faire plusieurs choix :

  • Construire une base d’exploration sur laquelle il peut ajouter des modules :
    • Réparation de vaisseaux
    • Construction de vaisseaux
    • Stockage
    • Défenses
    • …
  • Récolter les ressources (s’il y a des vaisseaux équipés dans la flotte pour la récupération et pour le stockage)
  • Attaquer les flottes présentes (pirates & joueurs)
  • Attendre ici
  • Partir dans un autre trou de ver
  • Scanner le système pour avoir plus de détail
    • Si les scanner sont peu efficace, cela mobilise la flotte pour une longue durée (en fonction de la précision demandée) l’exposant ainsi à de forts risques

Attraits des secteurs sombres

  • Ils permettent potentiellement de récolter de grandes quantité de ressources difficilement trouvable dans l’univers connu
  • Ils permettent de dissimuler une partie de sa flotte ou de ses technologies dans des secteurs difficilement trouvable.
  • Les trous de vers se refermant rapidement, cela permet d’avoir une flotte de combat qui permette de faire des attaques surprise sur des systèmes très éloigné et donc potentiellement éviter des représailles.

Dangers des secteurs sombres

  • Lorsqu’on a amassé une grande quantité de ressources, si on veut les rapatrier dans sa base initiale, il y a potentiellement une très grande distance à parcourir.
  • Si un joueur peut faire des attaques surprise à partir d’un trou de ver, il peut aussi bien en subir
  • Si on s’est enfoncé loin dans les niveaux de systèmes sombres, il peut-être très long d’en sortir
  • Il n’y a pas de carte des systèmes sombre donc on peut s’y perdre, si par exemple on a pas laissé de vaisseau dans un système qui permette de trouver un accès vers l’univers connu, la base peut-être perdue
